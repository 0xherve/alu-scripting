# ALU Scripting

This repository contains scripting projects and exercises, focused on various programming and scripting concepts.

## Overview

This repository is part of the ALU (African Leadership University) curriculum, containing hands-on scripting exercises and projects to develop programming skills.

## Repository Structure

```
alu-scripting/
├── README.md
└── regular_expressions/
    ├── README.md
    └── 0-simply_match_school.rb
```

## Projects

### Regular Expressions
Located in the `regular_expressions/` directory, this project focuses on building and understanding regular expressions using Oniguruma (Ruby's regex engine).

**Files:**
- `0-simply_match_school.rb` - A Ruby script demonstrating basic regular expression pattern matching

## Requirements

- Ruby (for regular expression exercises)
- Text editor or IDE of your choice

## Usage

Each subdirectory contains its own README with specific instructions. To run the Ruby scripts:

```bash
cd regular_expressions
ruby 0-simply_match_school.rb
```

## Learning Objectives

- Understanding and implementing regular expressions
- Scripting fundamentals
- Pattern matching and text processing
- Building reusable code solutions
